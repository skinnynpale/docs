Все крутится вокруг двух сущностей: **пользователь** и **организация**.

## Создание пользователя и организации

При регистрации вы создаете сущность «Пользователь». По итогу регистрации у пользователя есть выбор: создать собственную организацию или дождаться приглашения в существующую организацию.

### Ограничения

- Один пользователь не может создавать более одной организации
- Исключением выступает Партнерский кабинет (подробнее в разделе «Партнерам»)
- Один пользователь может быть приглашен в несколько организаций

## Добавление пользователей в организацию

Для добавления пользователя в вашу организацию:

1. Перейдите в «Настройки организации»
2. Выберите подраздел «Пользователи»
3. Попросите пользователя предварительно зарегистрироваться по ссылке [https://platform.trysasha.ru/register](https://platform.trysasha.ru/register)
4. Введите его почту в появившемся поле
5. Укажите право доступа к организации
6. Нажмите на кнопку «Добавить пользователя»

<Warning>
На текущий момент удаление пользователей или организаций невозможно.
</Warning>

## Отслеживание пользователей

Для отслеживания всех активных пользователей организации перейдите в раздел «Настройки организации», подраздел «Пользователи».
