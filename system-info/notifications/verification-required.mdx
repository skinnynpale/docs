Для запуска звонков вам необходимо связаться с менеджером, подписать бумаги и оплатить одну из подписок.

## Что требуется для верификации

1. **Связь с менеджером** — обратитесь в службу поддержки для назначения персонального менеджера
2. **Подписание документов** — оформление необходимых юридических документов
3. **Оплата подписки** — выбор и оплата одного из тарифных планов

<Warning>
Без прохождения верификации запуск звонков невозможен
</Warning>
