Для подключения фоллоу-апов в WhatsApp через сервис Wazzup следуйте инструкции ниже.

## Создание канала связи

1. Перейдите в раздел «Интеграции» → «Каналы связи»
<img src="/images/integrations/communication-channels/whatsapp/1.png" />
2. Нажмите на кнопку «Создать новый»
<img src="/images/integrations/communication-channels/whatsapp/2.png" />
3. В всплывающем окне выберите тип канала связи «Wazzup»
<img src="/images/integrations/communication-channels/whatsapp/3.png" />
4. Нажмите на кнопку «Создать»
## Получение API ключа

1. Зайдите в свой личный кабинет в сервисе Wazzup
<img src="/images/integrations/communication-channels/whatsapp/4.png" />
2. Перейдите в раздел «Интеграции с CRM»
<img src="/images/integrations/communication-channels/whatsapp/5.png" />
3. Выберите вариант «API»
<img src="/images/integrations/communication-channels/whatsapp/6.png" />
4. Скопируйте API ключ
<img src="/images/integrations/communication-channels/whatsapp/7.png" />
5. Вставьте API-ключ в всплывающем окне на платформе
<img src="/images/integrations/communication-channels/whatsapp/8.png" />
## Получение ID канала

1. Вернитесь в личный кабинет Wazzup
<img src="/images/integrations/communication-channels/whatsapp/9.png" />
2. Перейдите в раздел «Каналы»
3. Найдите нужный канал и нажмите на значок справа
<img src="/images/integrations/communication-channels/whatsapp/10.png" />
4. У вас откроется поддомен выбранного канала
<img src="/images/integrations/communication-channels/whatsapp/11.png" />
5. В URL ссылки будет лежать ID данного канала — скопируйте его
<img src="/images/integrations/communication-channels/whatsapp/12.png" />
6. Вставьте ID канала в всплывающем окне на платформе
<img src="/images/integrations/communication-channels/whatsapp/13.png" />

## Завершение настройки

1. Добавьте номер телефона для тестирования интеграции и нажмите кнопку «Проверить»
<img src="/images/integrations/communication-channels/whatsapp/14.png" />
2. Если всё прошло успешно, нажмите на кнопку «Создать», если нет, проверьте правильность ввода ID и API ключа
<img src="/images/integrations/communication-channels/whatsapp/15.png" />
## Привязка к колл-листу

1. Перейдите в раздел «Колл-листы»
<img src="/images/integrations/communication-channels/whatsapp/16.png" />
2. Выберите нужный колл-лист
<img src="/images/integrations/communication-channels/whatsapp/17.png" />
3. Перейдите в раздел «Канал связи»
<img src="/images/integrations/communication-channels/whatsapp/18.png" />
4. Нажмите на кнопку «Привязать канал связи»
<img src="/images/integrations/communication-channels/whatsapp/19.png" />
5. Выберите ранее созданный канал связи
<img src="/images/integrations/communication-channels/whatsapp/20.png" />
## Готово!

Теперь при передаче лида в рамках этого колл-листа клиенту будет приходить закрывающее сообщение в WhatsApp.
<img src="/images/integrations/communication-channels/whatsapp/21.png" />

## Тарификация

<Note>
Закрывающее сообщение в WhatsApp отсылается с вашего аккаунта. 

**Генерация сообщения на стороне Sasha AI бесплатна** и не тарифицируется, все взаиморасчеты по стоимости отправки сообщения производятся между вами и сервисом Wazzup напрямую.
</Note>
