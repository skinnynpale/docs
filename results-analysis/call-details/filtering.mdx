## Фильтр по телефону

Используйте для поиска разговора с конкретным абонентом.

<Tip>
Введите полный номер телефона или его часть для быстрого поиска нужного контакта.
</Tip>

## Фильтр по договоренности

Используйте для отделения успешных и неуспешных разговоров.

<Note>
Этот фильтр поможет вам быстро оценить эффективность звонков и выделить перспективные контакты.
</Note>

## Фильтр по причине отказа

Помните как в разделе "Логика передачи лида" мы обсуждали, что у Саши есть свой РОП? 

Эта нейросеть помимо отделения успешных разговоров от неуспешных присваивает каждому диалогу, который не закончился договоренностью причину отказа.

### Перечень причин отказа

Перечень причин отказа статичен и не подлежит кастомизации с вашей стороны:

- **Неадекватная реакция**
- **Сброс после приветствия**
- **Секретарь**
- **Не целевой клиент**
- **Неаргументированный отказ**
- **Аргументированный отказ**
- **Продавец не убедил**

### Интеграция с CRM

На своей стороне, с помощью написания микросервисов и API документации вы можете настраивать кастомные сценарии взаимодействия Sasha AI и вашей CRM системы.

**Например:**

### Пользуйтесь из коробки

Система фильтрации работает автоматически и не требует дополнительной настройки с вашей стороны.
