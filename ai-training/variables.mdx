Обрабатывая базу клиентов хороший живой оператор смотрит на дополнительные поля в карточке клиента. Он запоминает о клиенте какие-то пункты и исходя из них корректирует стратегию работы с лидом.

Саша может также.

С помощью сущности «переменная» вы можете обогащать знания нейросети о каждом номере телефона.

## Как обогатить контекст Саши о каждом номере телефона в отдельности?

Перейдите в раздел «Скиллбейсы» и выберите нужную вам систему инструкций. Нажмите на нее и перейдите в пространство настройки.

### Шаг 1. Определите переменные

Определите, какой информацией вы планируете обогащать контекст Саши. Внутри себя присвойте каждому типу информации свою переменную в формате:

```
{variable}
```

**Например:**

```
{client_name}     // имя клиента
{client_business} // сфера деятельности клиента
{client_surname}  // отчество клиента
{client_history}  // история сделок клиента
```

### Шаг 2. Вставьте переменные в скиллбейс

После прототипирования названий переменных — вставьте их в нужные разделы вашего скиллбейса. Продолжайте руководствоваться здравым смыслом, подробнее об этом сказано в разделе «Без ТЗ — результат ХЗ».

### Шаг 3. Обогатите базу клиентов

Когда вы обогатите свой скиллбейс нужными переменными, не забудьте обогатить значениями этих переменных базу клиентов, которую вы загрузите в «колл-лист» и будете прозванивать с этим скиллбейсом.

Подробнее об этом читайте в разделе «Колл-листы — загрузка контактов — переменные»
